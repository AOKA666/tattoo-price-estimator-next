# Tattoo Price Estimator - Next.js MVP

## Includes
- Next.js app router structure
- Interactive estimator page
- Basic metadata SEO
- FAQ structured data (JSON-LD)
- CTA section for lead capture direction

## Run
```bash
cd tattoo-price-estimator-next
pnpm install
pnpm dev
```

Open: http://localhost:3000

## Notes
- Current pricing is heuristic/rule-based
- Best next steps: add lead form, city pages, analytics, and image upload scoring
